# Health Assistance System

## Description
A desktop application developed using Java, JavaFX, MySQL, and JDBC to manage patients, doctors, appointments, and health records.

## Features
- User authentication
- Patient management
- Doctor management
- Appointment scheduling
- MySQL database connectivity
- JavaFX graphical interface
- Multithreading for appointment reminders

## Technologies Used
- Java
- JavaFX
- MySQL
- JDBC

## How to Run
1. Clone the repository
2. Configure MySQL database
3. Open the project in IntelliJ IDEA or VS Code
4. Run the application
